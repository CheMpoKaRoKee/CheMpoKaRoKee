# Андрей — Python / Backend / AI Automation

Начинающий разработчик, который фокусируется на backend-разработке, автоматизации и практическом применении AI-инструментов.

Собираю проекты вокруг реальных сценариев: Telegram-боты, desktop-приложения, API-интеграции, GitHub-инструменты и помощники для разработки.

## Основной фокус

- Python backend
- FastAPI
- Telegram Bot API
- REST API-интеграции
- автоматизация рабочих процессов
- AI-инструменты для разработки

- работа с GitHub и документацией проекта

## Текущие проекты

### Learning Mentor

Desktop-приложение для самостоятельного изучения fullstack-разработки.

Что внутри:

- учебный маршрут по темам;
- генерация теории;
- проверочные задания;
- чат-наставник;
- сохранение прогресса;
- portable-режим для Windows.

Стек:

- Electron
- React
- Vite
- Tailwind CSS
- OpenRouter API
- pnpm
- electron-builder

### Telegram Task Assistant

Telegram-бот для задач, напоминаний и простых рабочих сценариев.

Цель проекта — сделать удобный интерфейс в Telegram для личной и небольшой командной работы: создавать задачи, отслеживать статусы, получать напоминания и связывать бот с внешними сервисами через API.

Что показывает проект:

- работу с Telegram Bot API;
- проектирование команд бота;
- интеграцию с внешними сервисами;
- хранение связей между пользователями, задачами и статусами;
- подготовку проекта к запуску на сервере.

### Automation Bot Lab

Репозиторий для экспериментов с ботами и автоматизацией.

Используется как площадка для проверки идей, команд, API-запросов и небольших сценариев, которые позже можно переносить в более структурированные проекты.
