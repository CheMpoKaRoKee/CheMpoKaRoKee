# Андрей — AI Automation / API Integrations / Telegram Bots

Интегратор и автоматизатор с фокусом на AI-инструментах, Telegram-ботах, API-связках и прикладных рабочих процессах.

Собираю решения, которые связывают сервисы между собой: Telegram, NocoDB, Google Sheets, внешние API, AI-модели и GitHub-инструменты. Цель — не писать код ради кода, а быстро собирать полезные внутренние инструменты, помощников и автоматизации.

## Проекты

### [Telegram Task Assistant](https://github.com/CheMpoKaRoKee/telegram-task-assistant)

Python-бот для задач, напоминаний и простых рабочих сценариев в Telegram.

Что внутри:

- создание задач из Telegram;
- назначение ответственных;
- inline-меню;
- напоминания по срокам;
- хранение задач и пользователей в NocoDB;
- Kanban-представление через NocoDB;
- пример интеграции с Google Sheets;
- подготовка к запуску на VDS через systemd.

Что показывает проект:

- Telegram как интерфейс для процесса;
- NocoDB как лёгкую базу и Kanban-доску;
- связку Python + API + расписания;
- практическую автоматизацию без тяжёлой инфраструктуры.

Стек:

- Python
- python-telegram-bot
- httpx
- pydantic
- python-dotenv
- NocoDB API
- Google Apps Script
- systemd

### [Learning Mentor](https://github.com/CheMpoKaRoKee/Mentor)

Desktop-приложение для самостоятельного изучения fullstack-разработки с AI-наставником.

Что внутри:

- учебный маршрут по темам;
- генерация теории;
- проверочные задания;
- чат-наставник;
- сохранение прогресса;
- portable-режим для Windows.

Что показывает проект:

- интеграцию с AI API;
- упаковку идеи в desktop-приложение;
- продуктовый сценарий вместо отдельного скрипта;
- работу с настройками, прогрессом и локальными данными.

Стек:

- Electron
- React
- Vite
- Tailwind CSS
- OpenRouter API
- pnpm
- electron-builder
